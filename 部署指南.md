# 📱 Instagram 主动截流工具 - 部署指南

## 🎯 三种使用方案

### 方案 1：本地 Web 服务（推荐）
最简单的方式，你只需要运行 `web_service.py`，然后在浏览器打开 http://localhost:5000

**运行方式：**
```bash
# 安装依赖（如果还没装）
pip install flask

# 启动 Web 服务
python3 web_service.py
```

然后在浏览器中访问显示的地址即可使用。

---

### 方案 2：打包成可执行文件（不想让别人看代码）
使用 PyInstaller 打包成 .exe（Windows）或 .app（Mac），别人直接运行，看不到源代码。

**打包方式：**
```bash
# 运行打包脚本
python3 build.py

# 按照提示选择打包方式
```

打包完成后，在 `dist/` 目录下会有可执行文件。

---

### 方案 3：部署到云服务器（所有人都能用）
你可以把它部署到云服务器，做成在线服务。推荐平台：
- Vercel（免费）
- Render（免费）
- 阿里云/腾讯云

---

## 🚀 快速开始

### 方式 1：直接运行 Web 服务
```bash
cd 2-截流
python3 web_service.py
```
然后在浏览器打开 http://localhost:5000

### 方式 2：打包成可执行文件
```bash
cd 2-截流
pip install pyinstaller
python3 build.py
```

---

## 🔒 代码保护

使用 PyInstaller 打包后：
- ✅ 源代码被编译成字节码，别人看不到
- ✅ 双击即可运行，无需安装 Python
- ⚠️ 高级黑客仍可能反编译，但对普通用户足够安全

## 💡 使用建议

1. **如果给朋友用** → 方案 2（打包成可执行文件）
2. **如果公司内部用** → 方案 1（本地 Web 服务）
3. **如果要做成产品** → 方案 3（云服务部署）
