# 📱 Instagram 主动截流工具（dev 版）

> 你开发的 Instagram 评论截流工具，本地模型 + 话术库智能兜底

## 🎯 功能特性

- 🤖 **本地模型**：Qwen/Qwen2-1.5B-Chat，首次自动下载
- 💾 **话术库兜底**：模型不可用时自动切换，保证稳定
- 🎨 **Web 界面**：精美网页版，操作简单
- 🌍 **双语支持**：英文 + 中文翻译
- 🔄 **备选话术**：提供多个备选方案

## 🚀 快速启动

### 网页版（推荐）

```bash
cd 2-截流
python3 web_service.py
```

**访问地址**：http://localhost:5005

### 命令行测试

```bash
cd 2-截流
python3 intercept_bot_local.py
```

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `web_service.py` | ✅ 你修改过的网页版（主要） |
| `intercept_flask.py` | 旧版网页 |
| `intercept_bot_local.py` | 命令行版 |
| `截流人设与话术.md` | 话术库配置 |
| `截流助手Prompt.md` | Prompt配置 |
| `目标账号与标签库.md` | 标签配置 |

## 🎨 使用方法

1. 输入帖子内容（可选）
2. 输入评论内容（必填）
3. 点击"✨ 生成截流话术"
4. 复制英文话术到 Instagram

## 💡 提示

- **首次运行**：会自动下载模型（约 1-3GB），请耐心等待
- **网络问题**：如 HuggingFace 访问慢，可考虑镜像
- **离线使用**：模型下载后支持离线运行
