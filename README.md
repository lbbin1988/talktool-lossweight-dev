# Instagram 主动截流工具 (v1.0.0)

基于 Qwen2-1.5B 本地模型的 Instagram 评论截流和 DM 对话工具。

---

## 功能特性

### 1. 评论截流
- 📝 输入帖子内容和评论
- 🎯 自动生成 3 条完全不同的截流话术
- 🔄 5 个角度随机选择，每次生成都有新变化
- 🌐 中英文双语显示（原文 + 中文翻译）

#### 话术角度
1. **理解共情** - 先理解用户情绪，建立共鸣
2. **实用建议** - 给出具体可操作的小建议
3. **积极鼓励** - 给用户打气，传递正能量
4. **姐妹分享** - 用亲密的语气交流
5. **轻松幽默** - 用轻松幽默的方式沟通

---

### 2. DM 对话
- 💬 模拟 Instagram 私信聊天
- 📍 用户消息在左侧（气泡）
- 📍 我的回复在右侧（紫色气泡）
- 🇨🇳 每条消息都显示中文翻译
- 🎯 自动生成合适的回复

---

### 3. 翻译优化
- 🤖 使用 Qwen 模型直接翻译
- ✅ 准确自然，不是简单的字典翻译
- 📝 支持英文 → 中文翻译

---

### 4. 性能优化
- 🚀 模型强制使用本地缓存，不联网
- ⚡ 服务启动时预加载，第一次请求就很快
- 📱 支持局域网访问（http://192.168.x.x:5005）

---

## 使用方法

### 启动服务
```bash
cd ins-talktool-lossweight-dev
python3 web_service.py
```

### 访问网页
打开浏览器访问：
- 本地：http://localhost:5005
- 局域网：http://192.168.x.x:5005

### 使用流程
1. **评论截流**
   - 输入帖子内容（可选）
   - 输入用户评论
   - 点击"生成"按钮
   - 获得 3 条不同的话术

2. **DM 对话**
   - 切换到"DM 对话"标签
   - 输入用户消息
   - 点击"发送"
   - 查看自动生成的回复（带翻译）

## 重要修改记录

### v1.0.0 (2026-05-23)
✅ **初版稳定版本发布**

#### 核心修改
1. **模型加载优化**
   - 强制使用本地缓存路径
   - 完全不联网，避免 SSL 错误
   - 简化加载逻辑，Mac 上更稳定

2. **评论截流重构**
   - 预定义 5 个完全不同的话术角度
   - 每次随机选择 3 条
   - 确保主回复和备选话术完全不同

3. **DM 页面修复**
   - 用户消息和回复位置调换
   - 每条消息带中文翻译
   - 气泡颜色优化（紫色气泡白色文字，白色气泡黑色文字）

4. **翻译优化**
   - 移除简单字典翻译
   - 使用 Qwen 模型直接翻译
   - 更准确自然

5. **性能优化**
   - 服务启动时预加载模型
   - 第一次请求无需等待
   - 提高响应速度

## 技术栈

- **后端**: Python 3, Flask
- **AI 模型**: Qwen/Qwen2-1.5B-Chat (本地)

- **框架**: Hugging Face Transformers, PyTorch

## 系统要求

- **操作系统**: macOS / Linux / Windows
- **Python**: 3.8+
- **内存**: 4GB+ (推荐 8GB+)
- **磁盘**: 3GB+ 可用空间（模型缓存）

---

## 注意事项

1. **模型缓存**
   - 模型已在本地缓存
   - 无需重新下载
   - 缓存位置: `~/.cache/huggingface/`

2. **网络要求**
   - 不需要联网（模型已在本地）
   - 仅浏览器访问本地服务

3. **性能建议**
   - 首次启动需要加载模型（约 10-30 秒）
   - 之后请求会很快（< 2 秒）
